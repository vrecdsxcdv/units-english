## 🚀 Зачем переименовали?

### Преимущества новых названий:

1. **Краткость** — `A1-A2` вместо `Начальный курс A1`
2. **Международный стандарт** — все понимают CEFR уровни
3. **Логичность** — показывает диапазон уровней, которые покрывает блок
4. **Консистентность** — единый формат для всех блоков
5. **Меньше места в UI** — короткие названия лучше смотрятся в sidebar

### Было

```
Начальный курс A1  ← Длинно
Базовый курс B1    ← Неоднозначно
Продвинутый курс B2 ← Избыточно
```

### Стало

```
A1-A2  ← Кратко и понятно
B1-B2  ← Показывает диапазон
B2-C1  ← Международный стандарт
```
